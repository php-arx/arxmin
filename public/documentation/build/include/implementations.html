Implementations

Thanks to many of Arxmin users, there are multiple implementations of the template for easy integration with back-end frameworks. The following are some of them:

  • Yii 2 by Tobias Munk
  • Laravel by Arjay Angeles
  • Symfony by Marc Bach
  • Rails Gem by Nicolas Besnard

Note: these implementations are not supported by Cherrypulp. However, they do provide a good example of how to integrate Arxmin into different frameworks. For the latest release of Arxmin, please visit our repository or website
